# Exercícios em Java

Este repositório reúne exercícios práticos em Java desenvolvidos durante meus estudos de lógica de programação e fundamentos da linguagem.

O foco é praticar conceitos essenciais antes de avançar para projetos maiores e aplicações com interface ou banco de dados.

---

## Conteúdos praticados

- Estrutura básica de um programa Java
- Entrada de dados com `Scanner`
- Estruturas condicionais (`if` / `else`)
- Operações matemáticas simples
- Verificação de números (par / ímpar)
- Cálculo de média de notas
- Conversão de temperatura
- Sistema de login simples (console)

---

## Organização

Os códigos estão organizados em pacotes, separando os exercícios por tipo de lógica, facilitando leitura e manutenção.

Exemplos:
- Exercícios matemáticos
- Exercícios de condição
- Exercícios com entrada de dados
- Simulações simples de sistemas

---

## Tecnologias utilizadas

- Java
- JDK 17+
- IntelliJ IDEA
